# HUBFIVE Landing

Single-page React site scaffolded com Vite para apresentar a HUBFIVE: hero, portfólio, avaliações, quem somos, pacotes, benefícios, ecossistema e formulário de lead.

## Como rodar
1. Instale Node.js (>= 18). Caso esteja em um ambiente sem Node pré-instalado, instale-o antes de seguir.
2. Instale dependências:
   ```bash
   npm install
   ```
3. Ambiente de desenvolvimento:
   ```bash
   npm run dev
   ```
4. Build para produção:
   ```bash
   npm run build && npm run preview
   ```

## Notas
- Paleta escura com realces em **#B7FD13**.
- Carrosséis horizontais com scroll suave para portfólio e avaliações.
- Seção de pacotes com destaque para o plano recomendado e CTA de ancoragem para o formulário de lead.
- Estrutura pronta para receber ativos em `public/assets` (logos, screenshots, favicon).
