# HUBFIVE site

Landing page de uma página com navegação suave destacando o ecossistema HUBFIVE, portfólio, avaliações e formulário de contato para software sob medida.

## Como executar

Abra o arquivo `index.html` direto no navegador ou sirva localmente com qualquer servidor estático (por exemplo, `python -m http.server 4173`).

## Principais seções
- Hero com CTA e destaque para squads e produtos plug-and-play.
- Portfólio e Avaliações em carrosséis independentes.
- Quem Somos, Pacotes, Benefícios e Ecossistema HUBFIVE.
- Formulário para solicitar software sob medida e rodapé com contatos.
