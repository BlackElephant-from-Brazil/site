# BlackElephant Website

Este é o repositório do site oficial da **BlackElephant**, uma empresa de tecnologia focada em sites, sistemas, aplicativos e automações com IA.

## Tecnologias

- HTML5 Semântico
- CSS3 (Variáveis, Flexbox, Grid, Animações)
- JavaScript (Vanilla - ES6+)

## Estrutura do Projeto

- `index.html`: Arquivo principal
- `css/`: Estilos globais e componentes
- `js/`: Lógica e interatividade
- `assets/`: Imagens e recursos

## Cores

- **Fundo**: #0a0a0a
- **Destaque**: #39FF14 (Neon Lime)
